Make sure python is installed properly, environment variable is set.

Install these packages for robot setup :

1. pip install robotframwork
2. pip install robotframework-seleniumlibrary
3. pip install selenium (this is optional, when you want to develop normal scripts in other framework )

4. Install or integrate intellibot @seleniumlibrary patched in the pycharm by going to this location :

File -> Settings -> Pluggins -> search for Intellibot in the marktplace



All keyswords of selenium Library you can find at this place :

https://robotframework.org/SeleniumLibrary/SeleniumLibrary.html
